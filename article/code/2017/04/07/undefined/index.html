谢尔宾斯基三角形

使用 golang 绘制一个带颜色的谢尔宾斯基三角形。

定义1

使用去掉中心法：

  1. 取一个实心的三角形（多数使用等边三角形）
  2. 沿三边中点的连线，将它分成四个小三角形
  3. 去掉中间的那一个小三角形
  4. 对其余三个小三角形重复 1~4 步骤

实现

三角形
